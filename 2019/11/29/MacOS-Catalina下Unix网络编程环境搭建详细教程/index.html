MacOS CatalinaдёӢUnixзҪ‘з»ңзј–зЁӢзҺҜеўғжҗӯе»әиҜҰз»Ҷж•ҷзЁӢ

еҸ‘иЎЁдәҺ 2019-11-29 | еҲҶзұ»дәҺ UnixзҪ‘з»ңзј–зЁӢеҚ·1еӯҰд№  | | йҳ…иҜ»ж¬Ўж•°:
| еӯ—ж•°з»ҹи®Ў: 1.4k | йҳ…иҜ»ж—¶й•ҝ ≈ 6

иҝҗиЎҢзҺҜеўғ

ж“ҚдҪңзі»з»ҹпјҡmacOS Catalina 10.15

зҪ‘дёҠжүҫзҡ„еӨ§еӨҡж•°ж•ҷзЁӢе°Өе…¶жҳҜй…ҚзҪ®зҺҜеўғиҝҗиЎҢжәҗз ҒйӮЈйҮҢпјҢзӣ®еүҚжІЎжңүзңӢеҲ°е…ідәҺCatalinaзҡ„иҜҰз»Ҷж•ҷзЁӢпјҢиҝҷйҮҢжҲ‘иҜҰз»Ҷи®°еҪ•дәҶиҮӘе·ұеңЁйғЁзҪІзҺҜеўғж—¶йҒҮеҲ°зҡ„й—®йўҳе’Ңи§ЈеҶіж–№жЎҲгҖӮ

жң¬ж–Үзӣ®ж Ү

жҲҗеҠҹеңЁCatalinaж“ҚдҪңзі»з»ҹдёӢй…ҚзҪ®гҖҠUnixзҪ‘з»ңзј–зЁӢеҚ·1гҖӢжҸҗдҫӣзҡ„жәҗд»Јз ҒпјҢе№¶жҲҗеҠҹи·‘иө·жқҘз¬¬дёҖдёӘж ·дҫӢзЁӢеәҸdaytimetcpcil

иҝҗиЎҢзҺҜеўғжҗӯе»ә

жҗӯе»әзҺҜеўғзј–иҜ‘иҝҗиЎҢdaytimetcpcil.c

е®ҳж–№й…ҚзҪ®иҜҙжҳҺ

еңЁд№ҰдёӯжҸҗдҫӣзҡ„жәҗд»Јз Ғж–Үд»¶еӨ№пјҢжңүдёӘREADMEж–Үд»¶пјҢйҮҢйқўиҜҰз»ҶжҸҸиҝ°дәҶеҰӮдҪ•ж“ҚдҪңжүҚиғҪй…ҚзҪ®жҲҗеҠҹе№¶иҝҗиЎҢиҝҷдәӣд»Јз Ғпјҡ

QUICK AND DIRTY

Execute the following from the src/ directory:

./configure # try to figure out all implementation differences

cd lib # build the basic library that all programs need
make # use вҖңgmakeвҖқ everywhere on BSD/OS systems

cd ../libfree # continue building the basic library
make

cd ../libroute # only if your system supports 4.4BSD style routing sockets
make # only if your system supports 4.4BSD style routing sockets

cd ../libxti # only if your system supports cd
make # only if your system supports XTI

cd ../intro # build and test a basic client program
make daytimetcpcli
./daytimetcpcli 127.0.0.1

й…ҚзҪ®1

жҢүз…§иҝҷдәӣж“ҚдҪңзҡ„дҫқж¬Ўжү§иЎҢе‘Ҫд»Өпјҡ

вҖңPermission deniedвҖқпјҢжҹҘзңӢж–Үд»¶дёӯеҸ‘зҺ°жІЎжңүжү§иЎҢжқғйҷҗгҖӮchmodж·»еҠ жқғйҷҗеҚіеҸҜгҖӮ

дҫқж¬Ўжү§иЎҢдёӢйқўдёҖз»„е‘Ҫд»ӨпјҢжІЎй—®йўҳгҖӮ

дёӢдёҖз»„е‘Ҫд»Өпјҡ

иҝҷйҮҢеҮәзҺ°дәҶй—®йўҳпјҡ

жҹҘйҳ…иө„ж–ҷеҗҺ^[1]^пјҢи§ЈеҶіж–№жі•еҰӮдёӢпјҡ

еңЁunpv13e/libfreeдёӢжүҫеҲ°inet_ntop.cж–Үд»¶жҠҠ#include жіЁйҮҠжҺүеҶҚиҜ•дёӢдёҠиҝ°е‘Ҫд»ӨгҖӮвҖҰ жҲҗеҠҹпјҒ

е°ҶжҲҗеҠҹз”ҹжҲҗзҡ„ libunp.a ж–Үд»¶пјҢеӨҚеҲ¶еҲ°usr/libж–Үд»¶еӨ№дёӯпјҢе№¶дё”е°Ҷunpv13eдёӢзҡ„config.hе’Ңдҝ®ж”№еҗҺзҡ„unpv13e/libдёӯзҡ„unp.hеӨҚеҲ¶еҲ°usr/includeж–Үд»¶еӨ№дёӯгҖӮ

йҰ–е…Ҳе°Ҷunp.hдёӯзҡ„#include вҖңвҖҰ/config.hвҖқдҝ®ж”№дёә#include вҖңconfig.hвҖқ пјҢеӣ дёәunp.hе’Ңconfig.hйғҪж”ҫе…Ҙusr/includeдёӯгҖӮ

й…ҚзҪ®2пјҡдёүжқЎcpе‘Ҫд»Ө

з„¶еҗҺжү§иЎҢдёӢеҲ—е‘Ҫд»ӨпјҡгҖҗиҝҷдёүжқЎе‘Ҫд»ӨжҳҜCatalinaзҺҜеўғдёӢйҒҮеҲ°зҡ„жңҖеӨҡзҡ„еқ‘пјҒпјҒгҖ‘

жү§иЎҢз¬¬дёҖжқЎе‘Ҫд»Өж—¶дјҡйҒҮеҲ°еҰӮдёӢй—®йўҳпјҡ

вҖӢ еҺҹеӣ пјҡroutlessжңӘе…ій—ӯ(macдёәйҳІжӯўз”ЁжҲ·иҝӣе…ҘеҶ…ж ёжЁЎејҸиҖҢи®ҫзҪ®)

вҖӢ и§ЈеҶіж–№жЎҲпјҡ

йҮҚеҗҜMacпјҢејҖжңәж—¶й•ҝжҢүcommand+Rиҝӣе…ҘжҒўеӨҚжЁЎејҸпјҢеңЁдёҠйқўзҡ„е®һз”Ёе·Ҙе…·дёӯжүҫеҲ°teriminal/з»Ҳз«ҜпјҢиҫ“е…Ҙ
csrutil disable
е…ій—ӯrootlessжңәеҲ¶;иҰҒеҶҚејҖеҗҜиҫ“е…Ҙcsrutil enableеҚіеҸҜ

е…ій—ӯroutlessеҗҺдҫқж—§жңүй—®йўҳпјҡ

еҺҹеӣ пјҡиӢ№жһңеҚҮзә§Catalinaзі»з»ҹеҗҺпјҢе°Ҷзі»з»ҹж–Үд»¶е’Ңз”ЁжҲ·ж–Үд»¶еҲҶзҰ»пјҢзі»з»ҹж–Үд»¶йғҪиў«mountдәҶRead-onlyеҲҶеҢәпјҢйҳІжӯўзі»з»ҹж–Үд»¶иў«жҒ¶ж„ҸзҜЎж”№гҖӮ^[2]^

и§ЈеҶіж–№жЎҲпјҡ

  1. зҰҒжӯўSIPпјҲзі»з»ҹе®Ңж•ҙжҖ§дҝқжҠӨпјү
    е°ұжҳҜдёҠйқўе…ій—ӯroutlessзҡ„ж“ҚдҪңгҖӮ

  2. mountеҲҶеҢәдёәеҸҜеҶҷжЁЎејҸ

    гҖҗжіЁж„ҸгҖ‘пјҡиҝҷдёӘе‘Ҫд»ӨеңЁзі»з»ҹйҮҚж–°еҗҜеҠЁеҗҺеӨұж•Ҳ

йҮҚж–°жү§иЎҢдёҠйқўзҡ„дёүжқЎcpж“ҚдҪңеҚіеҸҜгҖӮ

жү§иЎҢдёӢеҲ—е‘Ҫд»ӨпјҢokгҖӮ

жӯӨж—¶жҲ‘д»ҘдёәпјҡеҸҜд»Ҙж„үеҝ«жү§иЎҢд№Ұдёӯзҡ„жәҗзЁӢеәҸе•Ұ

дҪҶжҳҜжҠҘй”ҷпјҡ

еҺҹеӣ жҳҜ/usr/includeдёӯе№¶жІЎжңүжҲ‘д»¬еҲҡеҲҡеӨҚеҲ¶зҡ„еӨҙж–Үд»¶гҖӮ

иҝҷйҮҢзҷҫеәҰжүҫеҲ°зҡ„ж–№жі•иҰҒдёҚжҳҜxcode-select --installиҰҒдёҚе°ұжҳҜ**sudo installer -pkg /Library/Developer/CommandLineTools/Packages/macOS_SDK_headers_for_macOS_10.14.pkg -target /**дҪҶжҳҜиҰҒдёҚжҳҜжІЎжңүж•ҲжһңпјҢиҰҒдёҚе°ұжҳҜжүҫдёҚеҲ°иҝҷдёӘзӣ®еҪ•гҖӮ

д»ҺиҝҷйҮҢеҸҜд»ҘзҹҘйҒ“пјҢMACеҚҮзә§дёәCatalinaеҗҺпјҢ/usr/includeе°ұдёҚеӯҳеңЁдәҶпјҢд»ҘеүҚзҡ„/usr/includeж”ҫеҲ°дәҶSDKжүҖеңЁзҡ„ең°еқҖдёӢпјҲеҸҜд»ҘйҖҡиҝҮxcrun --show-sdk-pathжҹҘиҜўпјүгҖӮ

жң¬й—®йўҳзҡ„и§ЈеҶіж–№жі•е°ұжҳҜжҠҠдёҠйқўдёүжқЎcpзҡ„зӣ®ж Үең°еқҖиҪ¬еҲ°sdkжүҖеңЁзҡ„includeзӣ®еҪ•дёӢеҚіеҸҜпјҢеҚіпјҡ

зј–иҜ‘иҝҗиЎҢжңҚеҠЎеҷЁзЁӢеәҸ

жӯӨж—¶makeеҗҺжү§иЎҢзЁӢеәҸжҳҫзӨәпјҡ

еҺҹеӣ пјҡиҺ·еҸ–ж—¶й—ҙзҡ„жңҚеҠЎеҷЁпјҲport:13пјүжІЎжңүиҝҗиЎҢпјҢintro/дёӢжңүдёӘdaytimetcpsrv.cж–Үд»¶еңЁеҸҰдёҖдёӘз»Ҳз«ҜзӘ—еҸЈдёӢmakeеҗҺиҝҗиЎҢиҜҘжңҚеҠЎеҷЁзЁӢеәҸеҚіеҸҜпјҢеҰӮдёӢпјҡ
