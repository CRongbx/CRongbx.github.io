树莓派突然无法连接手机热点的解决方案

发表于 2019-11-28 | 分类于 树莓派 | | 阅读次数:
| 字数统计: 172 | 阅读时长 ≈ 1

问题描述

​ 原先我使用树莓派的方式一直都是将树莓派和Mac一起连接手机开放的热点。上一次使用时还运行流畅，这次打开树莓派后发现死活连不上手机热点。

解决方法

​ 将手机WLAN热点的AP频段从“5GHz频段”设置为“2.4 GHz频段”。设置完后，不到3秒就连接成功。

疑惑

​ 我翻阅了下树莓派3B+的硬件配置信息，上面明确写道它配置的网卡支持“5/2.4 GHz”两个频段，不知道为什么这次5GHz频段不能使用？

​ 如果有大佬知道原因，麻烦告知~
