C标准流输入输出函数-fputs fgets

发表于 2019-12-19 分类于 C语言 阅读次数： Valine：

​ 本文总结下C语言标准库文件<stdio.h>中关于流输入输出函数。

fputs

把C字符串str写到流stream中。

参数：

  • str: C string with the content to be written to stream.
  • stream: Pointer to a FILE object that identifies an output stream.

返回：成功 非负数； 出错返回EOF，并置位error indicator

fgets

从流stream（包括标准输入流stdin）获取字符串str，直到遇到EOF或newline结束。

参数：

  • str: Pointer to an array of chars where the string read is copied.
  • num: Maximum number of characters to be copied into str (including the terminating null-character).缓冲区大小
  • stream: Pointer to a FILE object that identifies an input stream.
    stdin can be used as argument to read from the standard input.

返回： 成功读入一行 str指针；读入出错 NULL，并置位error indicator；读入时遇到EOF，返回NULL；

On success, the function returns str.
If the end-of-file is encountered while attempting to read a character, the eof indicator is set (feof). If this happens before any characters could be read, the pointer returned is a null pointer (and the contents of str remain unchanged).
If a read error occurs, the error indicator (ferror) is set and a null pointer is also returned (but the contents pointed by str may have changed).

ps: 如何查看error indicator?

ferror(stream)
