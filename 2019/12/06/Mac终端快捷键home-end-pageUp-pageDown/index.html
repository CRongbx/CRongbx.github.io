Mac终端快捷键home,end,pageUp,pageDown

发表于 2019-12-06 分类于 MacOS使用细节 阅读次数： 阅读次数： Valine：

MacOS上快捷键：跳到行首（home），跳到行尾（end），翻页（pageUp, pageDown）,上一行下一行，到顶部，到底部。

跳到行首：control + A

跳到行尾：control + E

到上一行：control + N

到下一行：control + P

到顶部： fn + ←（浏览网页时用）

到底部： fn + → （浏览网页时用）

上翻页： fn + ↑

下翻页： fn + ↓
