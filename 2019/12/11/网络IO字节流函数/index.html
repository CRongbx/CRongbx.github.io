​ 介绍TCP套接字编程时常用于网络端读写字节流套接字的三类函数（自己编写）：read 、writen和readline的功能和实现。

​ 同时穿插Unix编程环境中对文件描述符、read、write函数的介绍，size_t和ssize_t的对比。

​ 介绍这三个函数前先强调一点：网络中字节流套接字的read, write读写的字节数可能 << 请求data size.所以在网络编程时，需要多次从缓冲区中读写，直到取完所有的数据。

函数原型

file descriptor文件描述符

​ file descriptor通常是一个小的非负整数，内核用以标识一个特定进程正在访问的文件。

​ 当内核打开一个现有文件或创建并打开一个新文件时，它都会返回一个文件描述符。在读写该文件时，就用这个文件描述符。

ssize_t和size_t

size_t: 对象（如字符串）长度（无符号的）

ssize_t：返回字节计数的函数（带符号的）

上面的函数在声明时提到返回值-1代表出错，所以需要有符号的返回值类型ssize_t
